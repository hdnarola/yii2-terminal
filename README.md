# Yii2 Terminal
Running console commands from a browser

# Requirements 
* PHP 5.6 or higher
* Yii2 v.2.0.20 and newest
* [Yii2 Base](https://github.com/wdmg/yii2-base) module (required)
* [jQuery Terminal Emulator](https://github.com/jcubic/jquery.terminal) (required)
